# 🤖 AI Customer Feedback Analyzer

A high-performance, AI-powered tool designed to transform raw customer feedback into actionable business insights. Built with **Python**, **Gradio**, and the **Groq API (Llama 3.3 70B)**.

---

## 🌟 Key Features

- **Single Feedback Analysis:** Instant sentiment, emotion, and priority detection for individual reviews.
- **Batch Processing:** Analyze multiple reviews at once and get a summarized report with statistics.
- **Actionable Recommendations:** Beyond just sentiment, the AI suggests specific business actions.
- **Visual Sentiment Scoring:** Intuitive scoring from -1.0 to +1.0 for precise tracking.
- **Professional UI:** Clean, responsive interface built with Gradio's soft theme.

## 🛠️ Tech Stack

- **Language:** Python 3.12+
- **LLM:** Llama 3.3 70B (via Groq Cloud)
- **Framework:** Gradio (Web UI)
- **Environment:** python-dotenv (Security)
